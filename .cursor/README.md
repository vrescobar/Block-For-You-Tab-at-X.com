# MDC Cursor

Este directorio contiene el archivo de cursor MDC para el proyecto X.com For You Blocker. El cursor MDC ayuda a:

- Mantener un registro de los archivos del proyecto
- Documentar las funciones y componentes principales
- Rastrear las dependencias y requisitos
- Facilitar el mantenimiento y la colaboración

## Estructura

- `cursor.json`: Contiene la metadata del proyecto, registro de archivos y documentación
- `README.md`: Este archivo de documentación

## Uso

El archivo cursor.json se actualiza automáticamente cuando hay cambios significativos en el proyecto. Los desarrolladores pueden consultar este archivo para:

1. Entender la estructura del proyecto
2. Encontrar documentación específica de archivos
3. Verificar dependencias y requisitos
4. Revisar el estado actual del proyecto

Para mantener actualizado el cursor, por favor actualiza el archivo cuando hagas cambios significativos en el proyecto.
